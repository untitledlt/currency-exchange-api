# Currency Exchange API

## Getting Started

Clone repo:

```
git clone git@github.com:untitledlt/currency-exchange-api.git
```

Switch folder:

```
cd currency-exchange-api
```

Create environment file:

```
cp sample.env .env
```

_Set EXCHANGERATE_API_KEY in .env file!!!_

Install dependencies:

```
yarn
```

Run tests:

```
yarn test
```

Run dev:

```
yarn dev
```
